AFA Asociación del Fútbol Argentino

Sobre este sitio

Esta aplicación web fue desarrollada con fines prácticos para el curso de SysOne, academia Java.

La misma fue realizada con spring boot y angular, detalle de tecnologías utilizadas:

  • Backend
  • Maven
  • Java v.8
  • Spring boot v.2.4.2
  • Front
  • Angular

Cómo desarrolle la app

Dividí el desarrollo en varias partes: al comienzo armé a estructura básica del proyecto para que cada ABM funcione correctamente. Y una vez que estuvo todo terminado, realice el desarrollo de los DTOs (inicialmente hechos dentro del controller pero los moví a la capa de servicios), la idea fue simular procesos de entrega en los que se iba incrementando la funcionalidad.

Agregué la documentación, y una vez finalizados comencé con algunos flujos del front simplemente para avanzar un poco más con la app. Lista las tres tablas (Jugador, Equipo e Historia) y permite dar de alta jugadores
